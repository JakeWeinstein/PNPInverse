### **Re point 10** (Langmuir Picard formula not specified) — **Accept**


```
R_net  =  k_des · Γ
```

Expanding:
```
k_hyd · c_M · 10^(−ΔpKa) · (1 − Γ/Γ_max)  −  k_prot · c_H · Γ/δ_OHP  =  k_des · Γ
```

Let F₀ ≡ k_hyd · ⟨c_M · 10^(−ΔpKa)⟩, B ≡ k_prot · ⟨c_H⟩ / δ_OHP:
