### Re point 5 — `_perturbed_initial_guess` under-specified

**Accept.** Proper signature with explicit ctx:

```python
def _perturbed_initial_guess(
    ctx: dict,
    U_manuf: fd.Function,
    mesh: fd.Mesh,
    *,
    eps: float = 1e-3,
) -> fd.Function:
    """Build U_init = U_manuf + ε·smooth_perturbation(x,y), with
    Dirichlet BCs reapplied. Smooth perturbation vanishes at all
    essential-BC boundaries so we don't violate them at the IC.
    """
    indices = ctx['mixed_space_indices']
    x, y = fd.SpatialCoordinate(mesh)
    pert = fd.Constant(eps) * fd.sin(pi * x) * fd.sin(pi * y)

    U_init = U_manuf.copy(deepcopy=True)
    for i in range(ctx['n_species']):
        U_init.sub(i).interpolate(U_manuf.sub(i) + pert)        # NOTE: from U_manuf, not U_init
    U_init.sub(indices.phi_index).interpolate(
        U_manuf.sub(indices.phi_index) + pert
    )

    # Reapply Dirichlet BCs so essential constraints hold at the IC.
    for bc in ctx['bcs']:
        bc.apply(U_init)
    return U_init
```

The interpolation source is `U_manuf.sub(i)`, not `U_init.sub(i)`,
avoiding the fragile self-reference.

### Re point 8 — indicator helper ambiguous on quadrature

**Accept.** Helper signature now requires `degree` and uses
`fd.Constant(threshold)`:

```python
def _expr_indicator_measure(
    expr,
    threshold: float,
    *,
    mesh: fd.Mesh,
    degree: int,           # REQUIRED, no default
    comparison: str = "lt",
) -> float:
    """Measure of {x : expr(x) {<,>} threshold} via quadrature with
    explicit degree. Uses fd.Constant for the threshold to avoid
    UFL type coercion issues."""
    thr = fd.Constant(float(threshold))
    cond = fd.lt(expr, thr) if comparison == "lt" else fd.gt(expr, thr)
    dx_q = fd.dx(domain=mesh, degree=degree)
    return float(fd.assemble(
        fd.conditional(cond, fd.Constant(1.0), fd.Constant(0.0)) * dx_q
    ))


def _domain_volume(mesh: fd.Mesh, *, degree: int) -> float:
    """Domain volume computed at the same quadrature degree as
    _expr_indicator_measure to make ratios meaningful."""
    return float(fd.assemble(
        fd.Constant(1.0) * fd.dx(domain=mesh, degree=degree)
    ))
```

All call sites pass `degree=quad_degree` (or `degree=SRC_QUAD_DEGREE_FINAL`
once pinned). Pairs of indicator-measure / volume calls always use
matching degree.
