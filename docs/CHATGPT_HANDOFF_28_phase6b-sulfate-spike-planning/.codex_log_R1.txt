### 1.4 Log-rate BV residual (live)

From `Forward/bv_solver/forms_logc.py:411-462` (cathodic only shown):

```
log_r = ln(k0) + u_cat + Σ_factor [ power * (u_sp − ln c_ref) ]
        − α · n_e · η_clipped
R = exp(log_r)
```

So for R4e:

```
log_r_R4e = ln(k0_R4e) + u_O2(0,t)                          # O2 surface
          + 4 * (u_H(0,t) − ln C_HP_HAT)                     # H+ factor
          − 0.5 · 4 · η_clipped                              # α·n_e·η, η = (V_RHE − E°_R4e)/V_T
```

`η_clipped` is `(V − E_eq) / V_T` clipped to ±100 *before* the
`α·n_e` multiplication. `exponent_clip = 100` is the only
PC-trustworthy setting; `clip = 50` produces fictitious peroxide
current and must not be used.

`u_i = ln c_i_nondim`. So `u_H(0,t) − ln C_HP_HAT = ln(c_H_surf / c_H_ref)`.

For pH 10.6 surface (current Phase 6α 100 µm × 1e-18 case):
`c_H_surf / c_H_ref = 10^(4 − 10.6) ≈ 2.5·10^(−7)`.
R4e cathodic factor: `(2.5·10^(−7))^4 ≈ 4·10^(−27)`.
R2e cathodic factor: `(2.5·10^(−7))^2 ≈ 6·10^(−14)`.

This means the cathodic **concentration** factor is enormously
suppressed at surface pH 10.6, but the BV **overpotential** factor
`exp(α·n_e·|η|)` at deep cathodic V_RHE = −0.4 V is huge:
`η = (−0.4 − 1.23)/0.02569 = −63.4`, clipped at ±100, OK.
`exp(0.5·4·63.4) = exp(126.8) ≈ 4·10^54`.

So `R4e ∝ (k0_R4e) · 4·10^(−27) · 4·10^54 · c_O2_surf`. With
`k0_R4e = ratio · k0_R2e`, ratio = 1e-18, the rate ends up finite —
the magnitudes balance with overpotential dominating. The current
model is in a regime where cathodic concentration factor *is*
suppressed but overpotential exponential more than compensates.

   100	def build_water_ionization_terms(
   101	    *,
   102	    ctx: dict,
   103	    conv_cfg: dict,
   104	    z_vals,
   105	    u_h_unclamped,
   106	    u_h_unclamped_prev,
   107	    ci_h_clamped,
   108	    ci_h_prev_clamped,
   109	    R_space,
   110	    u_clamp: float,
   111	) -> WaterIonizationBundle:
   112	    """Build the OH⁻ closure expressions used by the proton-condition residual.
   113	
   114	    The H⁺ log-concentration expression (``u_h_unclamped``) is the
   115	    pre-clamp value; the same symmetric clamp ``[-u_clamp, u_clamp]``
   116	    applied to ``c_H = exp(u_h_unclamped)`` is applied to ``-u_H`` to
   117	    bound ``c_OH = Kw_eff · exp(-u_H)``.  Because the clamp is symmetric
   118	    about zero, ``clamp(-x) = -clamp(x)``, so passing the already-built
   119	    ``ci_h_clamped`` (which uses the same symmetric clamp) and dividing
   120	    is byte-equivalent — but the explicit ``exp(-u_clamped)`` form keeps
   121	    the algebra parallel to the existing ``ci`` pattern and avoids
   122	    a divide-by-near-zero at quadrature points where ``ci_h_clamped``
   123	    underflows to ``exp(-100)`` ≈ 3.7e-44.
   124	
   125	    Parameters
   126	    ----------
   127	    ctx
   128	        Firedrake context.  Mutated in place: ``ctx['water_ionization']``
   129	        is set to the returned bundle for downstream Q3 / diagnostics.
   130	    conv_cfg
   131	        ``solver_options['bv_convergence']`` sub-dict.  Reads
   132	        ``enable_water_ionization`` (must be True; caller gates),
   133	        ``kw_eff_hat``, ``d_oh_hat``, ``a_oh_hat``.
   134	    z_vals
   135	        Per-species charge list.  Used to locate the H⁺ index.
   136	    u_h_unclamped, u_h_unclamped_prev

   138	        time levels.  In ``forms_logc.py`` these are ``ui[h_idx]`` and
   139	        ``ui_prev[h_idx]``; in ``forms_logc_muh.py`` they are the
   140	        reconstructed ``μ_H − em·z_H·φ`` expressions.
   141	    ci_h_clamped, ci_h_prev_clamped
   142	        UFL expressions for ``c_H = exp(clamp(u_H))`` at the two time
   143	        levels.  Caller has already built these for the dynamic-species
   144	        residual; we reuse them in ``E = c_H − c_OH`` to keep the time
   145	        derivative consistent with the rest of the species.
   146	    R_space
   147	        ``firedrake.FunctionSpace(mesh, "R", 0)`` for the Kw_eff
   148	        Function.
   149	    u_clamp
   150	        Symmetric clamp magnitude (typically 100.0 in production).
   151	
   152	    Returns
   153	    -------
   154	    WaterIonizationBundle
   155	    """
   156	    h_idx = resolve_h_index(list(z_vals))
   157	
   158	    kw_eff_initial = float(conv_cfg.get("kw_eff_hat", 0.0))
   159	    d_oh = float(conv_cfg.get("d_oh_hat", 0.0))
   160	    a_oh = float(conv_cfg.get("a_oh_hat", 0.0))
   161	
   162	    if d_oh <= 0.0:
   163	        raise ValueError(
   164	            f"enable_water_ionization=True but d_oh_hat={d_oh!r} is non-positive; "
   165	            "set bv_convergence['d_oh_hat'] to the OH⁻ nondim diffusivity."
   166	        )
   167	    if kw_eff_initial < 0.0:
   168	        raise ValueError(
   169	            f"enable_water_ionization=True but kw_eff_hat={kw_eff_initial!r} "
   170	            "is negative; the continuation ladder may start at 0 but cannot "
   171	            "go below."
   172	        )
   173	
   174	    kw_eff_func = fd.Function(R_space, name="kw_eff_water_ionization")
   175	    kw_eff_func.assign(kw_eff_initial)
   176	
   177	    d_oh_const = fd.Constant(d_oh)
   178	    a_oh_const = fd.Constant(a_oh)
   179	
   180	    u_clamp_c = fd.Constant(float(u_clamp))
   181	    neg_u_clamp_c = fd.Constant(-float(u_clamp))
   182	
   183	    # c_OH = Kw_eff · exp(-u_H), with the same symmetric clamp on -u_H
   184	    # as on u_H.  Because the clamp is symmetric, clamp(-u_H) = -clamp(u_H)
   185	    # and we could equivalently write Kw_eff / ci_h_clamped — but the
   186	    # explicit exp form preserves the existing ``ci`` algebra structure.
   187	    neg_u_h_clamped = fd.min_value(
   188	        fd.max_value(-u_h_unclamped, neg_u_clamp_c), u_clamp_c
   189	    )
   190	    neg_u_h_prev_clamped = fd.min_value(
   191	        fd.max_value(-u_h_unclamped_prev, neg_u_clamp_c), u_clamp_c
   192	    )
   193	
   194	    c_oh_expr = kw_eff_func * fd.exp(neg_u_h_clamped)
   195	    c_oh_prev_expr = kw_eff_func * fd.exp(neg_u_h_prev_clamped)
   196	
   197	    e_var_expr = ci_h_clamped - c_oh_expr
   198	    e_var_prev_expr = ci_h_prev_clamped - c_oh_prev_expr
   199	
   200	    bundle = WaterIonizationBundle(
   201	        h_idx=int(h_idx),
   202	        kw_eff_func=kw_eff_func,
   203	        d_oh_const=d_oh_const,
   204	        a_oh_const=a_oh_const,
   205	        c_oh_expr=c_oh_expr,
   206	        c_oh_prev_expr=c_oh_prev_expr,
   207	        e_var_expr=e_var_expr,
   208	        e_var_prev_expr=e_var_prev_expr,
   209	    )
   210	    ctx["water_ionization"] = bundle
   211	    return bundle
