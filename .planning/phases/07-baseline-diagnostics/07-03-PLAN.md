    Create `scripts/studies/sensitivity_visualization.py` following codebase patterns.

    **Structure:**

    1. `SensitivityConfig` frozen dataclass:
       - sweep_factors: tuple = (0.5, 0.75, 1.0, 1.5, 2.0)
       - params_to_sweep: tuple = ("k0_1", "k0_2", "alpha_1", "alpha_2")
       - jacobian_h: float = 1e-5 (central FD step, codebase convention)
       - output_dir: str = "StudyResults/v14/sensitivity"
       - extended_voltage_min: float = -60.0 (Claude's discretion; extend further if convergence holds)
       - snes_max_it: int = 400 (more iterations for extreme voltages)
       - dt: float = 0.25 (smaller dt for convergence at extended voltages)
       - max_ss_steps: int = 200 (compensate for smaller dt)
       - verbose: bool = True

    2. `build_sweep_factors(factors: tuple | None = None) -> list[float]`:
       - Return list(factors) if provided, else [0.5, 0.75, 1.0, 1.5, 2.0]

    3. `build_extended_voltage_grid(v_min: float = -60.0) -> np.ndarray`:
       - Start with v13 cathodic grid: [-1, -2, -3, -4, -5, -6.5, -8, -10, -13, -17, -22, -28, -35, -41, -46.5]
       - Add extended points: -50, -55, -60 (and -65, -70, -75 if v_min allows)
       - Also include anodic/symmetric points: +5, +3, +1, -0.5
       - Unique and sort descending (for warm-start order)
       - Return as numpy array

    4. `build_perturbed_params(param_name: str, factor: float, true_params: dict) -> dict`:
       - Copy true_params
       - Multiply the named parameter by factor
       - Return new dict (immutable pattern -- do not modify original)

    5. `evaluate_iv_curve(params: dict, voltage_grid: np.ndarray, config: SensitivityConfig) -> tuple[np.ndarray, np.ndarray]`:
       - Build solver params via make_bv_solver_params with extended SNES options (snes_max_it=config.snes_max_it, dt=config.dt)
       - Call solve_bv_curve_points_with_warmstart with k0_values=[params["k0_1"], params["k0_2"]], alpha_values=[params["alpha_1"], params["alpha_2"]]
       - Extract total_flux and peroxide_flux arrays from results
       - Handle SNES failures: log warning, set NaN for failed voltage points
       - Return (total_flux, peroxide_flux)

    6. `compute_jacobian_row(eval_func, base_params_array: np.ndarray, h: float = 1e-5) -> np.ndarray`:
       - Central finite differences: (f(x+h) - f(x-h)) / (2h) for each parameter
       - eval_func takes a params array and returns a scalar (observable at one voltage)
       - Returns gradient array of length n_params

    7. `compute_full_jacobian(voltage_grid, true_params, config) -> np.ndarray`:
       - For each voltage point, compute Jacobian row for both total and peroxide current
       - Result shape: (2 * n_voltages, 4) where first n_voltages rows are d(total)/d(param), second n_voltages are d(peroxide)/d(param)
       - Uses compute_jacobian_row with h=config.jacobian_h

    8. `generate_sweep_plots(param_name, voltage_grid, sweep_results, config)`:
       - Two subplots: total current (top) and peroxide current (bottom)
       - One line per factor, labeled "0.5x", "0.75x", etc.
       - X-axis: voltage (eta), Y-axis: current
       - Title: f"Sensitivity to {param_name}"
       - Save as `sweep_{param_name}.png`

    9. `generate_jacobian_heatmap(voltage_grid, jacobian_matrix, param_names, output_dir)`:
       - Two heatmaps: one for total current sensitivity, one for peroxide current sensitivity
       - X-axis: parameters (k0_1, k0_2, alpha_1, alpha_2)
       - Y-axis: voltage points
       - Color: magnitude of d(observable)/d(parameter), use diverging colormap
       - Normalize per column (parameter) to show relative sensitivity across voltages
       - Save as `jacobian_heatmap.png`

    10. `write_sweep_csv(param_name, voltage_grid, sweep_results, output_dir)`:
        - Columns: voltage, factor_0.5_total, factor_0.5_peroxide, factor_0.75_total, ..., factor_2.0_total, factor_2.0_peroxide
        - Save as `sweep_{param_name}.csv`

    11. `write_jacobian_csv(voltage_grid, jacobian_matrix, param_names, output_dir)`:
        - Columns: voltage, d_total_d_k0_1, d_total_d_k0_2, d_total_d_alpha_1, d_total_d_alpha_2, d_peroxide_d_k0_1, ...
        - Save as `jacobian_heatmap.csv`
